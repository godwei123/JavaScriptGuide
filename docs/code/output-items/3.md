## 代码输出 - 3

```js
var a = 1;
(function a() {
  a = 2;
  console.log(a);
})();
```

::: details 答案

```txt
ƒ  a () {
      a = 2;
      console.log(a);
 }
```

立即调用的函数表达式（IIFE） 有一个 自己独立的 作用域，如果函数名称与内部变量名称冲突，就会永远执行函数本身；所以上面的结果输出是函数本身；

立即执行的函数表达式(IIFE)的函数名称跟内部变量名称重名后，函数名称优先，因为函数名称是不可改变的，内部会静默失败，在严格模式下会报错

:::
