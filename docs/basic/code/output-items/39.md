## 代码输出 - 39

```js
let a = { n: 1 };
let b = a;
a.x = a = { n: 2 };
console.log(a.x);
console.log(b.x);
```

::: details 答案

```txt
undefined
{n:2}

```

点号的运算优先级大于等于号
赋值操作从右向左
所以, a.x=a={n:2} 可以表示为 a.x=(a={n:2}). a.x 的值是先计算的,此时 a 的指向依然是对象{n:1}. a.x 的值是括号里的返回值,
所以是在对象{n:1}上添加了 x 这个属性,属性值为{n:2}. 然后是变量 a 被重新赋值,指向了对象{n:2}. 但变量 b 的指向依然是原先的对象.
所以, a.x 的值是 undefined, b.x 的值是{n:2}.
:::
